
# How to run

### 1. Clone the repo
``` shell
  git clone 
```
### 2. Install dependencies
``` shell
  yarn
```

### 3. Copy create accounts.json
``` shell
  cp accounts.json.example accounts.json
```
### 

### 4. Run the tests
``` shell
  yarn hardhat --network localhost test 
```
